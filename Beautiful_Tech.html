Beautiful Tech

Published on: 7-25-25

Sometimes tech is beautiful. I prefer software that does one thing very well. I like working in a bash terminal for that reason. In every command I execute, I can see the hard work and polish that went into it.

I can’t say I usually get the same feeling when I am using the tools of web development. So much of that space was created haphazardly that it almost feels arbitrary. But like everything under the sun, there are exceptions. I found an exception recently.

Let me give you an example:

            
            
            
            function helloWorld() {
                console.log("Hello, World!");
            }

I always assumed that to get code to display in a well-formatted block like the above, you needed a lot of extra plugins or Javascript code. It turns out it’s built into html and I love it. Hopefully, I will get to use it a lot more here.
